Instagram Login Page Clone

This project is a clone of the Instagram login page built using HTML and CSS.
It is created for frontend practice and to improve UI design skills.

🚀 Features

* Clean and modern UI
* Responsive design
* Styled using pure CSS
* Beginner-friendly project

🛠️ Tech Used

* HTML5
* CSS3

📂 How to Use

1. Clone the repository
2. Open index.html in your browser

This project is part of my web development practice to improve frontend skills.
